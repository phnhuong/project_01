### 9.3 Animation Guidelines

**DO:**
- âœ… Use subtle transitions (150-300ms)
- âœ… Animate transform and opacity
- âœ… Provide instant feedback
- âœ… Use `prefers-reduced-motion`

**DON'T:**
- âŒ Animate width/height (causes reflow)
- âŒ Use overly long animations (>500ms)
- âŒ Auto-play animations
- âŒ Animate on every interaction

```css
/* Respect user preferences */
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```
